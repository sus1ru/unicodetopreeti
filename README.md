# Unicode to Preeti Converter (Python)

This project provides a simple and reusable Python module to convert Nepali Unicode text into Preeti font encoding.

It works by sending input text to an online conversion service and extracting the converted Preeti output from the response. The module is designed to be used both as a standalone script and as an importable package in other Python projects.

## Features

- Convert Unicode Nepali text → Preeti encoding
- Supports both string input and file input
- Clean, reusable class-based design
- Easy integration into existing Python/Django projects
- Uses online converter for accurate results

## Use Cases

- Legacy font conversion (Unicode → Preeti)
- Data migration for old systems
- Batch text processing
- Integration into backend services or APIs
